# YouTube Music 桌面版

桌面環境上開啟並使用 YouTube Music 的簡單包裝程式。

## 功能
- 直接在桌面視窗內瀏覽 YouTube Music（自動隱藏選單列）。
- 支援彈出視窗與重新整理（Ctrl/Cmd + R）。

## 直接安裝
- 從 GitHub Releases 下載 `YouTubeMusicSetup.exe` 後執行即可安裝：<https://github.com/JeremySu0818/YouTube-Music/releases>

## 從原始碼建置
```bash
git clone https://github.com/JeremySu0818/YouTube-Music
cd YouTube Music
npm install   # 開發相依套件
npm run make  # 編譯可執行檔案
```
